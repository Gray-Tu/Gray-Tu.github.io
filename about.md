## 關於我

嗨，我是 Gray Tu。

這裡記錄我在生物資訊、資料分析與軟體開發的學習筆記和實作心得，也會寫一些日常隨筆，分享生活中的想法和觀察。

## 文章分類

- **生物資訊** — 單細胞分析、基因體學、生物資料處理
- **筆記** — 技術學習筆記、工具使用心得
- **日常隨筆** — 生活雜感、閱讀心得、隨想記錄

## 這個部落格

本站使用 [Jekyll](https://jekyllrb.com/) 建置，託管於 [GitHub Pages](https://pages.github.com/)。

### 如何新增文章

在 `_posts/` 資料夾中新增 Markdown 檔案，檔名格式：

```
YYYY-MM-DD-your-post-title.md
```

文章開頭加上 front matter：

```yaml
---
title: "文章標題"
categories: [分類名稱]
tags: [標籤1, 標籤2]
---
```

可用的分類：`生物資訊`、`筆記`、`日常隨筆`（也可自行新增）。

推上 GitHub 就會自動建置發布。
