這個時代太神奇了。

今天清明連假第二天，搭車的空閒時間，我把一個廢棄快八年的 GitHub Page 整個重構了一遍。從 Hexo 預編譯的靜態殘骸，變成乾淨的 Jekyll 架構 — 有版面、有分類、有 Demo 頁，連 commit 版號都自動顯示在首頁。

整個過程最關鍵的一件事：**把需求釐清**。

只要我能講清楚「我要什麼」，Claude Code 的產出速度就快到不可思議。不用自己查 Jekyll 文件、不用翻 Liquid 語法、不用一個一個手刻 CSS。我只要專注在「我想要的東西長什麼樣子」，剩下的它來。

生命有限，時間是最稀缺的資源。以前這種事大概會被我放在 TODO list 裡再躺個八年，但現在，搭車的零碎時間就搞定了。

這不是偷懶，是把精力放在真正重要的地方 — 想清楚要做什麼，而不是怎麼做。

未來應該會更常這樣：有個念頭，釐清需求，然後讓 AI 幫忙把它變成現實。

省下來的時間，拿去過生活。
